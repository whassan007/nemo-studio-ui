# Contributing to NeMo Studio UI

We welcome contributions to NeMo Studio UI! Here's how you can help.

## How to open issues
Please use the GitHub issue tracker to report bugs, request features, or ask questions.

## How to submit pull requests
1. Fork the repository on GitHub
2. Clone your fork locally
3. Create a branch for your changes
4. Make your changes and commit them
5. Push your branch to your fork
6. Submit a pull request against `whassan007/nemo-studio-ui`

## Code style guidelines
- Follow standard formatting for React and TypeScript (Prettier & ESLint are used in the frontend wrapper if built on it later).
- Use clear and descriptive variable names.
- Document any complex logic.

## Areas where contributions are welcome:
* Additional component definitions and parameter schemas
* Backend integration adapters (NeMo API, HuggingFace API)
* Accessibility improvements
* Localization (Arabic, French, Japanese)
* Educator mode and classroom tooling
